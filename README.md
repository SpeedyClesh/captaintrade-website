# CaptainTrade — Personal Portfolio

> Developer · AI Prompt Engineer · Crypto & FX Trader

Live site: **[captaintrade.netlify.app](https://captaintrade.netlify.app)**

---

## About

This is my personal portfolio and landing page. I built it from scratch to represent three things I do seriously — software development, AI engineering, and financial trading. The goal was a site that feels like a Bloomberg terminal and a developer portfolio at the same time. Clean, fast, no frameworks, no fluff.

---

## Built With

- **HTML5** — semantic, accessible structure
- **CSS3** — custom properties, grid, flexbox, animations
- **Vanilla JavaScript** — no dependencies, no bloat
- **Formspree** — contact form email delivery
- **Netlify** — hosting, CI/CD, security headers

No frameworks. No build tools. No npm. Just clean, hand-written code that loads fast and works everywhere.

---

## Features

- Live market ticker tape (Crypto & FX pairs)
- Scroll-triggered reveal animations
- Animated stat counters
- Interactive terminal block
- Fully functional contact form
- Mobile-responsive with burger menu
- WebP image optimization with JPG fallback
- SEO-optimized with JSON-LD structured data
- Open Graph + Twitter Card meta tags
- Security headers via Netlify config

## Local Development

No build step required. Clone the repo and open `index.html` directly in your browser, or use Live Server in VSCode for hot reload.

```bash
git clone https://github.com/SpeedyClesh/captaintrade-website.git
cd captaintrade-website
# Open index.html in browser or use VSCode Live Server
```

---

## Deployment

Deployed on Netlify with automatic CI/CD. Every push to `main` triggers a redeploy.

```bash
git add .
git commit -m "your message"
git push
# Netlify picks it up automatically
```
