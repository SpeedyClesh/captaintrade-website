# CaptainTrade — Personal Website

## Project Structure
```
captaintrade-website/
├── index.html          ← Main HTML (all sections)
├── css/
│   └── style.css       ← All styles (dark terminal theme)
├── js/
│   └── main.js         ← Interactions, animations, counter
├── assets/
│   └── images/         ← Add your images here
└── README.md
```

## How to Run Locally
1. Open the folder in VSCode
2. Install the **Live Server** extension (by Ritwick Dey)
3. Right-click `index.html` → **Open with Live Server**
4. Site opens at `http://127.0.0.1:5500`

## What to Customize
- Replace `@CaptainTrade` social handles with your real ones
- Add real project screenshots to `assets/images/`
- Update the project cards with your actual projects
- Swap placeholder links (`href="#"`) with real URLs
- For the contact form: sign up at formspree.io and replace the form action
